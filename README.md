# Getting Started

## Clone The Repository

```bash
git clone https://github.com/ValdryanIvandito/connect-ethereum-wallet.git
```

## Go to the Directory

```bash
cd connect-ethereum-wallet
```

## Install Dependencies:

```bash
yarn
```

_or_

```bash
npm i
```

## Run The Development Server

```bash
yarn dev
```

_or_

```bash
npm run dev
```
